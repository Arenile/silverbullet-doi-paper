# DOI Paper Import

Import research paper metadata from a DOI or arXiv ID and create a structured note with frontmatter.

Supports: ACM Digital Library, IEEE Xplore, arXiv, and any CrossRef-registered publisher.

## Install

Use the **Library: Install** command with this page's URL, or manually copy the `DOI Paper` page into your space under `Library/doi-paper/`.
